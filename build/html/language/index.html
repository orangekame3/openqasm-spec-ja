言語リファレンス¶

OpenQASM 3.0の言語仕様について詳しく解説します。

概要¶

OpenQASM 3.0は、量子回路とクラシカル計算の両方をサポートする包括的なプログラミング言語です。

主要な機能¶

  • 量子ビットとクラシカルビットの操作

  • パラメータ化されたゲート

  • 条件分岐と繰り返し処理

  • 関数定義とモジュール化

  • リアルタイムクラシカル計算
